## ⚡ About Me
I'm a 2nd-year Computer Science undergrad focused on **Data Engineering** and **Backend Architecture**. I prefer building robust, scalable systems and data pipelines over wrestling with abstract, math-heavy machine learning models. 

Right now, I am actively building complex systems and aiming to be fully internship-ready for Summer 2026. 

**Currently working on:**
* **Data Pipelines:** Engineering Python-based ETL workflows to process and analyze heavy datasets (like global banking data).
* **Backend Architecture:** Designing RESTful APIs and database schemas using Node.js, Express, and PostgreSQL (recently built the backend for *KAPS MedTech* and the *TripSync* platform).
* **System Optimization:** Transitioning towards distributed data processing and messaging streams using Apache Kafka and Spark.
